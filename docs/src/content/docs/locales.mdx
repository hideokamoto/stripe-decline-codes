## Available Locales

| Locale Code | Language | Status |
|-------------|----------|--------|
| `en` | English | <Badge text="Supported" variant="success" /> |
| `ja` | Japanese | <Badge text="Supported" variant="success" /> |

## Usage

Pass the locale code as the second argument to `getDeclineMessage` or `formatDeclineMessage`:

```ts
import { getDeclineMessage } from 'stripe-decline-codes';

getDeclineMessage('insufficient_funds', 'en'); // English (default)
getDeclineMessage('insufficient_funds', 'ja'); // Japanese
```

When a locale is not specified, `'en'` is used as the default.

## Fallback Behavior

If a translation is not available for the requested locale, `getDeclineMessage` returns `undefined`. Use the English message as a fallback when needed:

```ts
import { getDeclineMessage } from 'stripe-decline-codes';

function getLocalizedMessage(code: string, locale: string): string {
  return (
    getDeclineMessage(code, locale as 'en' | 'ja') ??
    getDeclineMessage(code, 'en') ??
    'An error occurred with your payment.'
  );
}
```

## Accessing Raw Translations

For advanced use cases, you can access the underlying data directly:

```ts
import { DECLINE_CODES } from 'stripe-decline-codes';

const info = DECLINE_CODES['insufficient_funds'];
console.log(info.translations?.ja?.nextUserAction);
// => "別のお支払い方法を使用してもう一度お試しください。"
```
